# Gestion des applications {#manage-applications}

Dans les déploiements d’expérience **une** application représente le service ou le produit que vous souhaitez contrôler à l’aide d’indicateurs de fonctionnalité. Avant de pouvoir créer des indicateurs de fonctionnalité, vous devez intégrer au moins une application à la console.

## Qui peut gérer les applications ? {#permissions}

Seuls les membres dotés du rôle **Admin** peuvent ajouter ou modifier des applications. Si vous ne disposez pas de ce rôle, contactez votre administrateur.

## Ajouter une application {#add-application}

Pour ajouter une application, consultez [Intégration de votre application](onboard-your-application.md) pour obtenir des instructions détaillées.
