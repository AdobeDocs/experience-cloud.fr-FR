# Sélectionnez votre sandbox {#sandbox}

Les déploiements d’expérience reposent sur Adobe Experience Platform. Avant d’utiliser les indicateurs de fonctionnalité, sélectionnez la sandbox qui correspond à votre environnement actuel, comme vous le feriez dans toute autre application Adobe Experience Platform.

## Comment sélectionner un sandbox {#how-to}

Utilisez le sélecteur de sandbox dans la barre de navigation supérieure de la console Déploiements d’expérience pour sélectionner le sandbox approprié avant de créer ou de modifier des indicateurs de fonctionnalité.
