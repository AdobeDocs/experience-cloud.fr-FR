# Applications de bureau {#desktop-applications}

Les applications de bureau s’intègrent aux déploiements d’expérience via SDK. Contactez votre représentant Adobe pour obtenir des conseils d’intégration spécifiques à votre pile de bureau.

>[!NOTE]
>
>La documentation de Desktop SDK est en cours de préparation et sera bientôt disponible.

## ID de l’application {#app-id}

Les clients de bureau peuvent utiliser un **code de produit et version du produit** comme identifiant de l’application.
